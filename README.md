# Data-Sharer
Sistema de compartición de archivos para el INCAN. Este sistema está diseñado para compilarse y ejecutarse como una aplicación de escritorio. La compartición de archivos debe de hacerse de manera segura a través de un dominio con https. Este proyecto cifra y descifra los archivos que se transfieren de acuerdo a la operación realizada (carga o descarga). 

NOTA: Esta es una versión preliminar que no se encuentra en producción. Se comparte esta versión para fines de demostración. Este proyecto es un cliente para un sistema de almacenamiento que se encuentra desplegado en infraestructura propia del Cinvestav Unidad Tamaulipas (2026).
